# landing-v0

Landing page built with Next.js + Tailwind CSS.

## Quick start

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Tech
- Next.js
- Tailwind CSS

## Vercel v0
I used Vercel v0 to generate UI sections (Hero, Feature cards, Testimonials) and integrated the exported JSX into `components/`.
